# 绿化工人管理系统 (Qicai Landscape Workforce Management)

## 📋 项目简介

本系统是一个面向绿化工程行业的工人与司机考勤、工资管理平台。

**涉及端：**
- 🖥️ **Next.js 管理后台** — 超级管理员使用，负责项目管理、批次审核、工资结算
- 📱 **微信小程序（司机端）** — 司机使用，提交每日工人考勤、查看工资
- 📱 **微信小程序（管理员端，可选）** — 管理员移动端轻量操作

**核心角色：**
- 超级管理员（老板，1~2人）
- 司机（约10人）
- 工人（100+人，不直接使用系统）

## 🚀 快速开始

### 1. 环境准备

| 环境 | 版本要求 | 检查命令 |
|:---|:---|:---|
| JDK | 17+ | `java -version` |
| Maven | 3.9+ | `mvn -version` |
| MySQL | 8.0+ | `mysql --version` |
| Redis | 7.0+ | `redis-server --version` |
| Node.js | 20 LTS+ | `node -v` |
| pnpm | 8.x+ | `pnpm -v` |

> ⚠️ 如果以上命令无法识别，请先安装对应环境并配置系统环境变量。

### 2. 初始化数据库

```bash
# 登录 MySQL 并执行初始化脚本
mysql -u root -p < sql/init-schema.sql
```

默认管理员账号：`admin` / `123456`

### 3. 阅读设计文档

1. [`docs/系统架构设计.md`](docs/系统架构设计.md) — 了解整体架构、接口规划、前端规范
2. [`docs/数据库设计说明.md`](docs/数据库设计说明.md) — 理解表关系、关键设计决策、索引设计
3. [`sql/init-schema.sql`](sql/init-schema.sql) — 查看完整数据库结构

## 🏗️ 技术栈概览

### 后端
- **Spring Boot 3.2+** + Java 17
- **MyBatis-Plus**（ORM）
- **JWT**（无状态认证）
- **Redis**（缓存）

### 管理后台
- **Next.js 14+**（App Router）
- **TypeScript** + **Tailwind CSS**
- **shadcn/ui**（UI 组件库）
- **Zustand**（状态管理）
- **React Hook Form** + **Zod**（表单校验）

### 微信小程序
- **原生微信小程序** 或 **Taro 3.x**
- 大按钮、少跳转、高对比度设计（适配中老年司机）

## 📌 核心业务规则速览

| 规则 | 说明 |
|:---|:---|
| 工人考勤 | 一个工人每天只有一条考勤记录（数据库唯一索引保证）|
| 司机考勤 | 司机每天全勤，工资 = 基础日薪 + 加班工资 |
| 批次审核 | 司机提交批次 → 管理员审核通过（不驳回）→ 状态锁定 |
| 工资计算 | 审核时固化工资到冗余字段，避免重复计算 |
| 工资结清 | 支持按年结算，允许部分结清，通过 `is_settled` 标记 |
| 项目分配 | 审核时必须给工人考勤分配项目 |
| 日志记录 | 所有操作记录到 `operation_logs`，工资发放也通过日志记录 |
